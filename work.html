myFlix

myFlix allows users to view information about different movies including Description, Genre and Director. Users can sign up for an account and add favorite movies to their profile. I It utilizes password hashing to encrypt users’ passwords. This project was built using the MERN stack (MongoDB, Express, ReactJS, Nodejs) and styled with Bootstrap.
